Rock Auto Shopping Cart Design

About the Project

Role: UX/UI Designer

Tools Used: Adobe XD

This is a redesign of an existing company called Rock Auto. The original website is quite outdated, and their shopping cart is integrated on the same page as the homepage. The checkout is also on the same page so there is a lot going on. I wanted to redesign this with more simplicity, create an easier to use interface, and update the colors to be more chic. Adobe XD was used to wireframe this page.
